# CodeAlpha_car-price-prediction
Machine Learning project for predicting car selling prices using random forest regression(machine learning algorithm)

## Project Overview
This project predicts the selling price of used cars using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Encoding Categorical Variables
5. Model Training
6. Prediction
7. Evaluation

## Model Used
Random Forest Regressor

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Dataset Features
- Present Price
- Car Age
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner
- Brand

## Results
The model successfully predicts car selling prices based on vehicle features.
